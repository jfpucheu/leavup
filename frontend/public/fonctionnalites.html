Calculs légaux automatiques

Leavup applique automatiquement les règles du Code du travail français. Vous n'avez pas besoin de connaître la loi — le logiciel s'en charge.

  • ✓ Jours ouvrés (25j/an) ou ouvrables (30j/an)
  • ✓ Acquisition progressive ou par anticipation
  • ✓ 11 jours fériés nationaux (avec Pâques mobile)
  • ✓ Fractionnement automatique (Art. L3141-19)
  • ✓ Alerte pour congé principal > 24 jours ouvrables
  • ✓ Période de référence civile ou légale (1er juin)

Conforme au Code du travail

Les calculs de Leavup sont basés sur les articles L3141-1 à L3141-32 du Code du travail. Acquisition, fractionnement, période de référence — tout est géré automatiquement selon la loi française.
